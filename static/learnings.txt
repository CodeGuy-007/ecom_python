run "python manage.py runserver" to start localhost (like webpack)

run "python manage.py startapp appname" to create a new app in root folder

Authentication is built into Django

run "python manage.py makemigrations" to import all the tables for user Authentication

then run "python manage.py migrate" to merge all the tables for user Authentication
